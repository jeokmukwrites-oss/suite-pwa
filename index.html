구글 드라이브 자동 동기화

연결하면 캘린더 기록과 런처 링크가 드라이브의 앱 전용 폴더에 자동 저장·복원됩니다(수동 가져오기 불필요). 권한은 이 앱이 만든 데이터에만 한정됩니다.

상태: 연결 안 됨
일회성 설정 방법 (펼치기)
  1. Google Cloud Console에서 프로젝트를 만듭니다.
  2. ‘API 및 서비스 → 라이브러리’에서 Google Drive API를 사용 설정합니다.
  3. ‘OAuth 동의 화면’을 외부/테스트로 만들고, 본인 지메일을 테스트 사용자로 추가합니다.
  4. ‘사용자 인증 정보 → OAuth 클라이언트 ID → 웹 애플리케이션’을 만들고, 승인된 자바스크립트 원본에 이 앱을 올린 주소(예: https://내사이트)를 넣습니다.
  5. 발급된 클라이언트 ID를 위 칸에 붙여넣고 ‘드라이브 연결’을 누릅니다.

※ 동기화는 인터넷과 고정 HTTPS 주소가 필요합니다(파일을 직접 열기(file://)에서는 동기화가 비활성화됩니다). 데스크톱 앱과의 연결은 기존 ‘내보내기/가져오기’를 함께 쓰시면 됩니다.
